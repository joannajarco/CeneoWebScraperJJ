PURPOSE OF THE APPLICATION

The main purpose of the application is to extract opinions about products from the Ceneo.pl website. For product, the code of which will be given as a parameter, its name and all available opinions about it should be extracted.

DESCRIPTION OF THE APPLICATION'S OPERATION

Using the extract page users can input a product code, and the application retrieves the corresponding product's name and all available opinions about it from Ceneo.pl. Each opinion is parsed to extract various details including opinion ID, author, recommendation, score, content, advantages, disadvantages, helpfulness ratings, publishing date, and purchase date. When the wrong product code is entered the error will pop up on the screen.
